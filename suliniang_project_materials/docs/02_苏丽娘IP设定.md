# 苏丽娘IP设定

## 角色定位

苏丽娘是从昆曲《牡丹亭》中走出的苏州文化精灵。她以杜丽娘的文化记忆为母体，但作为新的原创IP独立存在：温婉、灵秀、端庄、亲和，能用多语种带游客读懂苏州园林、昆曲、苏绣与文物。

## 命名解释

| 字 | 含义 |
|---|---|
| 苏 | 苏州，也呼应苏绣、苏式美学 |
| 丽娘 | 承接《牡丹亭》杜丽娘的昆曲根脉 |

合起来就是「苏州的丽娘」：既有传统来源，又可作为现代城市IP运营。

## 文化DNA

| DNA | 体现方式 | 价值 |
|---|---|---|
| 昆曲 | 杜丽娘原型、旦角眼妆、闺门旦头面、水袖 | 世界级非遗，文化分量足 |
| 苏绣 | 戏服上的牡丹、玉兰、缠枝花卉与精细金线 | 苏州工艺名片，可落到周边 |
| 园林 | 花窗、太湖石、回廊、水景、拙政园讲解 | 文旅场景直接可用 |

## 视觉铁律

这些是后续生图、视频、建模、周边打样都不能丢的识别点：

- 脸型：Q版萌系比例，圆润婴儿肥，小圆下巴，不瘦脸。
- 眼睛：细长上扬的吊梢丹凤眼，这是最高识别点，不要变成普通大圆眼。
- 眼妆：柔和红色眼影，带昆曲旦角妆感，端庄不妖。
- 头面：昆曲闺门旦头饰，点翠、绒花、珠泡子、红流苏，清雅不堆砌。
- 服饰：月白 + 淡青灰 / 竹青，白水袖，苏绣牡丹或玉兰纹样。
- 配色：月白、淡青灰、粉色、点翠蓝、流苏红、金线。
- 禁用：京剧式大红大金、黄绿底色、荧光色、妖媚表情。

## 人格口吻

苏丽娘说话应该像「懂文化、懂游客、温柔但不空泛的苏州姐姐」：

- 语速偏慢，有导览感。
- 中文可带一点古典气，但不能难懂。
- 英文和意大利语要清楚服务游客，不硬翻古文。
- 回答问题时先给直观答案，再补文化典故。
- 不做网红腔，不做过度卖萌。

## 衍生方向

已具备或可继续推进的形态：

- 角色基准立绘。
- 三视图与表情细节。
- 3D手办、潮玩盲盒。
- 剪纸、亚克力挂件、钥匙扣。
- 毛绒玩偶。
- 文具、丝巾、伴手礼。
- 文旅联名：苏州博物馆、园林景区、平江路、山塘街。

## 差异化

苏丽娘不是单纯的「文物拟人」或「古风少女」。她的优势是同时具备：

- 原创角色的可商用空间。
- 昆曲非遗的文化深度。
- 苏州园林的场景落地。
- 中 / 英 / 意三语数字导游的国际传播点。
- 周边开发和景区采购的转化可能。
