# 项目总览

## 一句话定义

「苏丽娘」是一个以昆曲《牡丹亭》杜丽娘为文化母体、面向苏州文旅场景的原创数字IP。当前资料包已把她推进到「中 / 英 / 意三语数字人导游」方向：能讲园林与昆曲、能回答游客问题、能定制路线、能做合影彩蛋，并可延展为文创周边与景区导览产品。

## 项目背景

- 项目面向：2026 苏州世界文创大赛 OPC 共创赛道。
- 截止时间：2026 年 7 月 15 日。
- 支持单位：WaytoAGI 苏州 OPC 社区。
- 赛事主题：用 AI 技术让文化活起来，让游客玩起来。
- 现有形态：本地网页 Demo + IP 形象资产 + 视频脚本 + 技术实施方案。

## 核心策略

1. 先用已有苏丽娘 IP 资产起跑，避免从零做人设。
2. 用同一套 IP 同时支撑「AI文化IP」与「AI数字人导游」两类表达。
3. 用中 / 英 / 意三语讲解扣住「世界文创」的差异点。
4. 用网页 Demo 证明互动性，用周边形态证明赛后转化价值。

## 当前完成度

- IP：角色名称、文化母体、人格、视觉规范、三视图、表情细节和周边形态基本成型。
- 产品：网页 Demo 已有六段体验，包含欢迎、定制路线、拙政园游览、景点讲解、问答、合影。
- 技术：纯 Python + 单页前端的本地演示已跑通；问答和路线生成接 DeepSeek；合影生成有兜底图。
- 内容：已有 6 段网页讲解视频脚本和一版约 60 秒参赛短片脚本。
- 待补：数字人对口型视频、真实景点素材、报名规则核实、最终提交物包装。

## 我们要守住的四把锁

| 锁 | 解释 | 我们怎么体现 |
|---|---|---|
| OPC共创 | 方法可复用、可开放二创 | 把「现成IP转数字导游」做成社区可复制方法论 |
| 世界文创 | 有国际传播能力 | 中 / 英 / 意三语讲苏州，昆曲本身是世界级非遗 |
| 互动体验 | 不是只看视频，要能玩 | 语音问答、三语切换、景点选择、路线定制、合影 |
| 赛后转化 | 可展示、可采购、可卖货 | 文博会展示、景区小程序、IP周边、买家对接 |

## 内部定位

后续不要把它讲成「一个网页」。更好的说法是：

> 这是一个以苏州世界级文化符号为核心、由 AI 生成与交互能力驱动的文旅IP样板：前端是可体验的数字人导游，后端是可复用的城市文化IP生成与运营方法。
