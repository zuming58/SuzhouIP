# 数字人导游产品方案

## 产品定位

一个可演示、可扩展的本地网页 Demo：让评委或游客通过苏丽娘完成一段苏州园林数字导览体验。它不是完整商业系统，但已经能证明「数字人形象 + 多语种讲解 + 智能问答 + 文旅互动」的产品雏形。

## 目标用户

| 用户 | 场景 | 核心诉求 |
|---|---|---|
| 大赛评委 | 现场或线上看 Demo | 快速理解亮点，看到互动性和转化空间 |
| 入境游客 | 游览苏州园林 | 多语种讲解、能问能答、文化不隔膜 |
| 国内游客 | 扫码游园 | 趣味讲解、路线建议、合影打卡 |
| 景区 / 文旅采购方 | 看项目落地可能 | 可运营、可扩展、可接小程序或导览屏 |

## 当前 Demo 六段体验

### 1. 开场迎宾

苏丽娘立绘登场，提供中 / 英 / 意三语欢迎词。用户可以选择「定制行程」或「直接进园」。

### 2. 定制行程

用户填写游玩天数、同行人、偏好和特别要求，系统通过大模型生成个性化路线。网络异常时回退到经典路线，保证现场演示不空。

### 3. 进入拙政园

展示拙政园导览图和五个景点卡片：远香堂、与谁同坐轩、小飞虹、香洲、待霜亭。

### 4. 景点讲解

点击景点弹窗显示大图、名称和三语讲解词。视频字段已预留，后续放入数字人讲解视频即可自动播放。

### 5. 问问苏丽娘

支持语音输入和文字问答，由大模型扮演苏丽娘回答。断网时根据关键词使用预设 FAQ 兜底。

### 6. 合影彩蛋

生成「游客与苏丽娘在拙政园」的合影图。生图失败时显示预生成示例图，确保演示完整。

## 产品亮点

- 三语切换贯穿全站，不只是字幕包装。
- 交互路径完整，评委可以从欢迎到问答、合影走完一轮。
- 数据集中在内容文件中，后续加景点、改讲解词、加视频都较轻。
- 预留视频字段，数字人视频一旦完成可直接嵌入。
- 兜底逻辑完整，适合现场网络不稳定的比赛环境。

## 商业化路径

1. 景区讲解小程序：按景点二维码触发苏丽娘讲解。
2. 文博会互动屏：现场选择景点、问答、三语切换。
3. 城市文旅数字人：扩展到平江路、山塘街、苏博、虎丘等场景。
4. 文创周边销售：把数字IP和实体商品绑定。
5. 入境游服务：以英文和意大利语导览作为差异化入口。
