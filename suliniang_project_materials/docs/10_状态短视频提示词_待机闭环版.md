# 状态短视频提示词：待机闭环版

## 使用规则

参考图建议：

- 参考图 1：`character_3view_front_side_back.png`，用于锁定角色设计。
- 参考图 2：`character_idle_clasped_front_master_generated.png`，用于锁定正面待机首帧。

所有视频都遵守：

- 不使用“水袖”这个词，统一写“white sleeves / 白色衣袖”。
- 第一帧必须是参考图 2 的待机姿态。
- 最后一帧也必须回到参考图 2 的待机姿态。
- 动作只发生在中间 40%-60% 时间段。
- 角色比例、眼睛、服装、头饰、镜头距离、脚底线保持不变。
