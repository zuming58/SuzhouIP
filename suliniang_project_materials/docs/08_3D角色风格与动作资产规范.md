# 3D角色风格与动作资产规范

## 定版结论

当前苏丽娘已经形成一套稳定的 3D Q版角色风格。后续网页、PPT、视频、数字人口播、表情状态和周边展示，应优先使用 `assets_3d_character/` 里的这套资产，而不是继续混用早期 2D 插画版本。

这一版的关键词是：

> 精致 3D 潮玩感、昆曲闺门旦、江南雅色、端庄灵秀、可爱但不幼稚。

## 风格特征

### 角色比例

- Q版大头小身，约 2.5 到 3 头身。
- 头部体量大，身体纤细但不瘦削。
- 整体像高精度手办 / 盲盒角色，适合数字导游和文创周边双用。

### 面部

- 脸型圆润，婴儿肥明显，下巴小圆。
- 眼睛是细长上扬的丹凤眼，眼尾挑起，是核心识别点。
- 眼妆为柔和粉红 / 珊瑚红晕染，带昆曲旦角气质。
- 眉心红色花钿固定保留。
- 嘴型小巧，表情变化靠眼神、头部倾斜、手势和袖口动作共同完成。

### 头饰

- 高盘发，黑发有明显 3D 发丝高光。
- 点翠蓝头面是视觉锚点，带金色镶边和珍珠。
- 两侧牡丹 / 绒花形成粉红与珊瑚红层次。
- 珍珠链、红色流苏、垂珠都很重要，能强化昆曲头面和精致感。

### 服饰

- 主色为米白 / 月白。
- 辅色为淡青绿、浅竹青。
- 裙摆和外襟有粉色牡丹、枝叶与细线绣纹。
- 腰间浅青绿蝴蝶结和正中红色长流苏是固定识别点。
- 白色水袖厚重柔软，适合做导览动作、遮口笑、转场待机。

### 材质与光感

- 3D 渲染干净、柔和、高亮但不塑料。
- 背景以纯白或浅灰为主，方便抠图、网页叠放、PPT排版。
- 画面不能变成写实真人，也不要走暗黑、厚重、赛博或浓烈舞台戏曲路线。

## 角色不可变元素

后续生成、修图或视频驱动时必须守住：

- 丹凤眼，不改成大圆眼。
- 点翠蓝头面 + 珍珠链。
- 两侧粉 / 红花饰。
- 眉心红花钿。
- 月白与淡青绿服装。
- 牡丹绣纹。
- 红色长流苏。
- 白水袖。
- 温婉端庄的气质。

## 开发调用建议

开发、状态切换和视频生成时，优先调用 `assets_3d_character_normalized/` 中的统一比例版本。原始图片保留在 `assets_3d_character/`，适合追溯和重新处理。

### 网页状态匹配

- 欢迎页：`assets_3d_character_normalized/pose_inviting_bow.png`
- 景点卡片或讲解弹窗：`assets_3d_character_normalized/pose_point_to_scene.png`
- 问答输入中：`assets_3d_character_normalized/pose_listening.png`
- 大模型生成中：`assets_3d_character_normalized/pose_thinking.png`
- 回答播放中：`assets_3d_character_normalized/expression_mouth_open.png`
- 成功 / 彩蛋：`assets_3d_character_normalized/pose_sleeve_laugh.png`
- 待机 / 过渡：`assets_3d_character_normalized/pose_arrange_sleeves.png`

### 视频制作匹配

- 开场：颔首邀请。
- 讲园林：抬手指景。
- 听游客问题：侧耳倾听。
- 讲到诗词典故：闭眼或执扇半遮。
- 轻松结尾：掩口轻笑。
- 口播对口型测试：张嘴 + 表情组图。

### PPT 匹配

- 封面：执扇半遮或颔首邀请。
- 角色设定页：三视图 + 细节设定板。
- 产品流程页：按欢迎、听、想、讲、笑排列动作图。
- 商业转化页：3D角色 + 周边图并排。

## 禁忌

- 不要把眼睛生成成普通大圆眼。
- 不要改成京剧凤冠或大红大金戏服。
- 不要去掉头面、花钿、流苏和水袖。
- 不要让服装颜色变成黄绿、深红、紫蓝或荧光色。
- 不要做过度性感、妖媚、夸张搞怪表情。
- 不要混入现代服装、赛博装、武侠盔甲或真人写实脸。
