# 视频与内容制作方案

## 两类视频

当前资料包里有两条视频线，要分清：

1. 网页 Demo 内嵌讲解视频：6 段，包含欢迎 + 5 个拙政园景点，每段约 20 秒。
2. 参赛主片：约 60 秒，可裁 30 秒版，用于展示苏丽娘从昆曲走向数字文旅IP的整体概念。

## 网页 Demo 内嵌视频

### 目的

让网页从「图文 + 问答」升级为「苏丽娘真的在讲」。这是评委最容易感知的提升。

### 片段清单

| 片段 | 用途 | 背景 |
|---|---|---|
| welcome | 首页迎宾 | 拙政园 / welcome-bg |
| yuanxiangtang | 远香堂讲解 | 远香堂图 |
| yushei | 与谁同坐轩讲解 | 扇形小轩图 |
| xiaofeihong | 小飞虹讲解 | 朱红廊桥图 |
| xiangzhou | 香洲讲解 | 旱船 / 画舫图 |
| daishuangting | 待霜亭讲解 | 小岛橘林图 |

### 制作规格

- 优先竖屏 9:16，适合手机扫码。
- 同时可导出横屏 16:9，用于现场大屏或 PPT。
- 每段 18 到 22 秒。
- 人物半身，主立绘锁脸。
- 字幕建议烧录中文，英文 / 意大利语可作为后续版本。
- 语气温婉亲切，像懂行的苏州导览员，不要太戏腔。

### 验收标准

- 六段是同一个苏丽娘，脸型、眼睛、头面不漂。
- 口型大体对上，不出现明显崩脸。
- 背景与景点匹配。
- 字幕可读。
- 文件命名清晰，后续能填入内容数据的 `video` 字段。

## 参赛主片

### 核心创意

昆曲《牡丹亭》里的杜丽娘化作苏州文化精灵「苏丽娘」；她用水袖唤醒园林、文物和非遗，并作为多语种数字导游，带世界游客读懂苏州。

### 60 秒结构

| 时间 | 内容 |
|---|---|
| 0-6s | 水墨苏州园林清晨，镜头穿过花窗，出现《牡丹亭》画卷 |
| 6-12s | 杜丽娘从画卷中走出，化为 Q版苏丽娘 |
| 12-20s | 水袖拂过，花窗、太湖石、回廊被唤醒 |
| 20-30s | 文物活化，如秘色瓷莲花碗泛光旋转 |
| 30-40s | 苏丽娘成为数字导游，面向中外游客讲解 |
| 40-50s | 周边形态快切：盲盒、手办、剪纸、挂件、毛绒 |
| 50-60s | 回到园林花窗前，定格主视觉和参赛口号 |

### 三语台词

中文：

> 欢迎来到苏州，我是苏丽娘，让我带你走进这座两千五百年的水城。

English:

> Welcome to Suzhou. I'm Su Liniang. Let me walk you through this 2,500-year-old water town.

Italiano:

> Benvenuti a Suzhou. Sono Su Liniang, lasciate che vi guidi in questa antica città d'acqua.

## 内容包装建议

视频里一定要显性出现：

- 苏丽娘不是普通古风角色，而是昆曲 + 苏绣 + 园林的复合IP。
- 她会多语种讲解，尤其是意大利语。
- 她能互动，不只是播片。
- 她能变成周边和景区产品，有赛后转化。

## 制作优先级

1. 先做 10 秒对口型测试。
2. 再做首页 welcome 视频。
3. 批量做 5 个景点视频。
4. 做 60 秒参赛主片。
5. 最后剪一个 15 秒预告版，方便社媒传播。
