# Suliniang Project Materials

这是我们消化后的「苏丽娘数字人导游 / 苏州世界文创大赛」项目资料夹。

原始交接包的信息已经被重新整理为我们自己的工作资料：保留方向判断、产品逻辑、IP设定、技术路线、视频脚本和执行清单；去掉交接口吻、个人机器路径和密钥信息。

## 快速阅读顺序

1. `docs/00_项目总览.md`：先看项目是什么、为什么值得做、现在到哪一步。
2. `docs/01_大赛与赛道判断.md`：看赛事要求、创作方向、奖励和待确认事项。
3. `docs/02_苏丽娘IP设定.md`：角色定位、视觉铁律、文化DNA、周边方向。
4. `docs/03_数字人导游产品方案.md`：网页 Demo 的六段体验和可落地卖点。
5. `docs/04_技术路线与实现地图.md`：数字人、问答、三语、合影、兜底的技术拆法。
6. `docs/05_视频与内容制作方案.md`：参赛短片和网页内嵌讲解视频的制作口径。
7. `docs/06_素材资产清单.md`：已经挑出来的参考图和用途。
8. `docs/07_下一步执行清单.md`：按优先级推进。

## 资产目录

`assets_reference/` 里放的是后续最常用的参考图：赛事海报、苏丽娘主立绘、三视图、表情细节图、Demo 背景/导览图、周边形态参考。

`assets_3d_character/` 里放的是最新定版的 3D 苏丽娘角色资产：三视图、表情组、细节设定板，以及可直接用于开发调用的独立姿势/表情图。

`assets_3d_character_normalized/` 里放的是已经统一画布、人物比例和脚底基线的开发调用版。做网页状态切换和状态短视频时优先用这个目录。

## 我们的判断

这个项目最强的不是单个网页，而是「原创IP + 三语数字导游 + 可共创可转化」的组合。后续材料、视频、PPT 都应该围绕四个关键词展开：OPC共创、世界文创、互动体验、赛后转化。

最新视觉口径以 3D 定版资产为准，详见 `docs/08_3D角色风格与动作资产规范.md`。
